# CSD SOLO 挖矿 - 多合一安装脚本使用说明

## 📦 脚本功能

这是一个完整的多合一安装脚本，包含以下功能：

1. ✅ 自动检查系统环境和GPU
2. ✅ 自动安装所有依赖（Rust、CUDA等）
3. ✅ 自动创建项目文件夹 `~/csd-solo-miner`
4. ✅ 自动下载和编译 CSD 挖矿程序
5. ✅ 自动生成配置文件
6. ✅ 支持低带宽模式（只使用一个最优节点）
7. ✅ 在脚本内直接配置钱包地址

## 🚀 使用方法

### 第一步：配置钱包地址

编辑脚本 `solo-miner-install.sh`，修改钱包地址：

```bash
# 找到这一行并修改为您的钱包地址
WALLET_ADDRESS="YOUR_WALLET_ADDRESS_HERE"
```

改为：

```bash
WALLET_ADDRESS="your_actual_wallet_address_here"
```

### 第二步：运行安装脚本

```bash
chmod +x solo-miner-install.sh
./solo-miner-install.sh
```

### 第三步：启动挖矿

安装完成后，会在 `~/csd-solo-miner` 目录下创建完整的项目结构：

```bash
cd ~/csd-solo-miner
./start-miner.sh
```

## 📁 项目结构

安装后的目录结构：

```
~/csd-solo-miner/
├── bin/                          # 可执行文件
│   └── compute-substrate         # 挖矿程序
├── config/                       # 配置文件
│   └── miner-config.toml        # 挖矿配置
├── data/                        # 区块链数据
├── logs/                        # 日志文件
├── compute-substrate/           # 源码目录
├── start-miner.sh              # 启动脚本
├── stop-miner.sh               # 停止脚本
├── status.sh                   # 查看状态
└── view-logs.sh                # 查看日志
```

## 🎮 管理命令

进入项目目录后，可以使用以下命令：

```bash
cd ~/csd-solo-miner

# 启动挖矿
./start-miner.sh

# 停止挖矿
./stop-miner.sh

# 查看挖矿状态
./status.sh

# 查看实时日志
./view-logs.sh
```

## ⚙️ 配置说明

### 钱包地址
在脚本顶部配置：
```bash
WALLET_ADDRESS="your_wallet_address"
```

### 引导节点
可以添加多个引导节点：
```bash
BOOTSTRAP_NODES=(
    "/ip4/35.223.117.16/tcp/30333/p2p/12D3KooWEyoppNCUx8Yx66oV9fJnriXwCcXwDDUA2kj6vnc6iDEp"
    "/ip4/35.245.161.243/tcp/30333/p2p/12D3KooWHdiAxVd8uMQR1hGWXccidmfCwLqcMpGwR6QcTP6QRMuD"
)
```

### 安装目录
默认安装到 `~/csd-solo-miner`，可修改：
```bash
INSTALL_DIR="$HOME/csd-solo-miner"
```

## 🔧 低带宽模式

脚本已启用低带宽优化：
- 自动选择延迟最低的节点
- 只使用一个最优节点连接
- 减少网络带宽消耗

## 📊 GPU 挖矿

脚本会自动检测 NVIDIA GPU：
- 如果检测到 GPU 和 CUDA，将启用 GPU 挖矿
- 如果没有 GPU，将使用 CPU 模式

## ⚠️ 注意事项

1. **首次运行**：需要编译代码，可能需要 10-30 分钟
2. **磁盘空间**：确保有足够空间存储区块链数据（建议 50GB+）
3. **后台运行**：建议使用 screen 或 tmux 运行
4. **日志文件**：会自动保存在 `logs/` 目录

## 🛠️ 故障排除

### 问题：编译失败
```bash
# 更新 Rust 工具链
rustup update stable
rustup target add wasm32-unknown-unknown
```

### 问题：GPU 未检测到
```bash
# 检查 NVIDIA 驱动
nvidia-smi

# 安装 CUDA Toolkit
# 访问: https://developer.nvidia.com/cuda-downloads
```

### 问题：连接节点失败
- 检查网络连接
- 确认引导节点地址正确
- 查看日志文件了解详细错误
