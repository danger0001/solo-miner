# CSD SOLO 挖矿 - 使用说明 v4.0.0

## 📦 功能特性

这是一个全自动的一键安装脚本，包含以下功能：

1. ✅ 自动检查系统环境和GPU
2. ✅ 自动安装所有依赖（Rust、CUDA等）
3. ✅ 自动创建 `~/solo` 项目文件夹
4. ✅ 自动下载和编译 CSD 挖矿程序
5. ✅ 自动生成配置文件
6. ✅ 支持低带宽模式（只使用一个最优节点）
7. ✅ **运行时交互式输入钱包地址**

## 🚀 快速开始

### 一键安装

只需一行命令：

```bash
curl -fsSL https://raw.githubusercontent.com/danger0001/solo-miner/main/install.sh | bash
```

### 安装过程

1. **创建目录**：自动创建 `~/solo` 目录

2. **输入钱包地址**：安装过程中会提示您输入钱包地址：

```
=========================================
  请输入您的 CSD 钱包地址
=========================================

提示: 钱包地址通常以 0x 开头

钱包地址: [输入您的钱包地址]

您输入的钱包地址是:
0x1234567890abcdef...

确认无误? (y/n): y
```

3. **自动安装**：等待安装完成（首次运行需要编译，约10-30分钟）

### 启动挖矿

安装完成后：

```bash
cd ~/solo
./start.sh
```

## 📁 项目结构

安装后的目录结构：

```
~/solo/
├── bin/                      # 可执行文件
│   └── compute-substrate     # 挖矿程序
├── config/                   # 配置文件
│   └── miner-config.toml    # 挖矿配置
├── data/                    # 区块链数据
├── logs/                    # 日志文件
├── compute-substrate/       # 源码（编译后可删除）
├── start.sh                # 启动挖矿
├── stop.sh                 # 停止挖矿
├── status.sh               # 查看状态
└── view-logs.sh            # 查看日志
```

## 🎮 管理命令

进入项目目录后，可以使用以下命令：

```bash
cd ~/solo

# 启动挖矿
./start.sh

# 停止挖矿
./stop.sh

# 查看挖矿状态
./status.sh

# 查看实时日志
./view-logs.sh
```

2. 手动编辑 `config/miner-config.toml` 文件

## 🔧 低带宽模式

脚本已启用低带宽优化：
- 自动选择延迟最低的节点
- 只使用一个最优节点连接
- 减少网络带宽消耗

## 📊 GPU 挖矿

脚本会自动检测 NVIDIA GPU：
- 如果检测到 GPU 和 CUDA，将启用 GPU 挖矿
- 如果没有 GPU，将使用 CPU 模式

## ⚠️ 注意事项

1. **首次运行**：需要编译代码，可能需要 10-30 分钟
2. **磁盘空间**：确保有足够空间存储区块链数据（建议 50GB+）
3. **后台运行**：建议使用 screen 或 tmux 运行
4. **日志文件**：会自动保存在 `logs/` 目录

### 后台运行示例

使用 screen：
```bash
# 创建新会话
screen -S solo-miner

# 启动挖矿
cd ~/solo && ./start.sh

# 按 Ctrl+A 然后按 D 离开会话

# 重新连接
screen -r solo-miner
```

使用 tmux：
```bash
# 创建新会话
tmux new -s solo-miner

# 启动挖矿
cd ~/solo && ./start.sh

# 按 Ctrl+B 然后按 D 离开会话

# 重新连接
tmux attach -t solo-miner
```

## 🛠️ 故障排除

### 问题：编译失败
```bash
# 更新 Rust 工具链
rustup update stable
rustup target add wasm32-unknown-unknown
```

### 问题：GPU 未检测到
```bash
# 检查 NVIDIA 驱动
nvidia-smi

# 安装 CUDA Toolkit
# 访问: https://developer.nvidia.com/cuda-downloads
```

### 问题：连接节点失败
- 检查网络连接
- 确认引导节点地址正确
- 查看日志文件了解详细错误

### 问题：想修改钱包地址
```bash
# 方法1: 编辑配置文件
cd ~/solo
nano config/miner-config.toml

# 方法2: 重新运行安装
curl -fsSL https://raw.githubusercontent.com/danger0001/solo-miner/main/install.sh | bash
```
